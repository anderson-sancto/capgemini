
Descrição do projeto:

 O projeto consiste em um sistema de cadastro de anúncios.
Após a inserção dos dados é possível obter relatório constante de número de visualizações,
compartilhamentos e cliques,baseados no valor de investimento indicado no ato do cadastro.
É possível filtrar relatório por nome do cliente e data.

Requisitos para utilização:

Python 3.8.6


Instrução:

Inserção de dados conforme solicitação e atenção ao formato da data, a mesma deve ser inserida observando o seguinte modelo ex: 12/01/2021
